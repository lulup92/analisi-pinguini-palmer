# Analisi Esplorativa dei Pinguini di Palmer 🐧

Questo progetto analizza le caratteristiche morfologiche dei pinguini dell'arcipelago di Palmer.

## Cosa ho fatto:
- **Data Cleaning**: Rimozione valori nulli e gestione anomalie (valori errati nella colonna sesso).
- **Feature Engineering**: Conversione delle unità di misura in KG e CM per una migliore leggibilità.
- **Visualizzazione**: Creazione di grafici statistici (KDE, Boxplot, Scatter, Hexbin) per identificare i pattern tra le specie.

## Tecnologie usate:
- Python
- Pandas
- Matplotlib / Seaborn
